Required and optional API keys for full Smart Planner functionality
===============================================================

1) YANDEX_MAPS_API_KEY
- Purpose: frontend map rendering (map picker in event form, routes view).
- Where to get: Yandex Developer Dashboard -> Yandex Maps JavaScript API key.
- Env vars:
  - backend: YANDEX_MAPS_API_KEY
  - frontend build/runtime: NEXT_PUBLIC_YANDEX_MAPS_API_KEY

2) YANDEX_ROUTER_API_KEY
- Purpose: real route duration/distance instead of mock provider.
- Where to get: Yandex Routing API / Distance Matrix API key.
- Env var: YANDEX_ROUTER_API_KEY

3) YANDEX_GEOCODER_API_KEY
- Purpose: address autocomplete, geocoding, reverse geocoding for text input and map point.
- Where to get: Yandex Geocoder API key.
- Env var: YANDEX_GEOCODER_API_KEY

Notes
-----
- If Yandex keys are not set, backend falls back to mock/stub behavior.
- For production, use separate keys and quotas for Maps, Routing and Geocoder services.
